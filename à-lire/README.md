# 📂 Articles à lire

Bienvenue dans le dossier **`a-lire/`** du repository **PFE Papers Summary**.  
Ce sous-dossier contient **tous les articles que nous prévoyons de lire**.

---

## 🧩 Que trouve-t-on dans ce dossier ?

- Les fichiers Markdown (`.md`) correspondant à chaque article identifié mais **non encore lu ou résumé**.
- Chaque fichier contient :
  - Le titre de l’article
  - La référence ou le lien vers l’article
  - L’objectif de lecture
